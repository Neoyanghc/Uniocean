# Uniocean

PyTorch Implementation of Uniocean

## Requirements

* Python >= 3.6 (3.7 recommended)
* PyTorch (tested with PyTorch 1.11.0)
* Optional: CUDA (tested with CUDA 11.3)
* numpy
* pandas
* scikit_learn
